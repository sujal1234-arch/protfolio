## ğŸš€ Getting Started

### Option 1: Simple HTTP Server (Recommended)

1. Open the project folder in your terminal/command prompt
2. If you have Python installed, run:

## ğŸ¯ Customization Guide

### Personal Information
Edit `index.html` to update:
- Your name in the hero section
- About me text
- Skills and technologies
- Project details
- Social media links
- Contact email

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-gradient: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    --accent-gradient: linear-gradient(135deg, #4facfe 0%, #00f2fe 100%);
    /* Customize these gradients to your preference */
}
```

### Projects
Update the project cards in `index.html` with:
- Project title and description
- Technology tags
- GitHub and live demo links

### Resume
Replace the resume button link with your actual PDF:
```javascript
// In script.js, find the download-resume event listener
window.open('path/to/your-resume.pdf', '_blank');
```
